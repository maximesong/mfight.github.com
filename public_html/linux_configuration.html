Fedora命令自动补全

用惯了ubuntu的apt-get，用fedora的yum时没有了自动补全，非常不习惯。后来才知道，fedora相比ubuntu，少了一个叫bash-completion的东西。那就装上呗。

sudo yum install bash-completion
编辑/etc/.bashrc文件，添加: 
if [ -f /etc/bash_completion ]; then
. /etc/bash_completion
fi
如果修改的是～/.bashrc，则需要执行 
source /etc/bash_completion
大功告成，试试看Tab的自动补全吧。
