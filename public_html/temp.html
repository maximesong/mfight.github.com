top使用WordPress搭建自己的blog

我的 blog 就是选用 WordPress 自己搭建的。主要原因如下：

  • 比较流行，Templates 和 Plugins 比较多，而且源码公开，可以按照自己的要求随便修改；
  • 自己比较懒，不想总是登录网页写 blog，而且在网页上写受限较多，编辑和保存都不方便；
  • 自己搭建blog，各种文档数据都可以很方便的操作和存档，随心所欲；
  • WordPress 只需 PHP + MySQL 就可以搭建，空间比较容易找，正好学校提供了。

我搭建过程中的一些经验会记录在 这里 。当然，非常欢迎去我的 blog 看看哦！
